Разработка программы "органайзер".

Программа предназначена для хранения, просмотра и редактирования данных по клиентам. 
+1. Использовать консольный ввод/вывод. 
+2. Информация по клиенту: 
-Уникальный код клиента в органайзере;
 	- Ф.И.О.; 
- должность; 
- наименование организации; 
- e-mail; 
- телефон (список).
 
+3. Для хранения данных использовать файл XML. 

Команды: 
+4. help - вывод справки по командам органайзера; 
+5. insert - добавить нового клиента; 
+6. update - редактировать; 
+7. delete - удалить; 
+8. list - вывести список клиентов
+9. list fieldName1;fieldName2;....;fieldNameN - вывести список клиентов. поля, по которым идет сортировка, не обязательно для ввода; 
+10. find - поиск клиента по номеру телефона.
 
+11. Использовать язык программирования -  Java.
+12. Покрыть код unit – тестами.
+13. Разработка приложения с использованием ИСР IntelliJ IDEA приветствуется.
